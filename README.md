# Biodigestores Villavicencio 🌿

Sitio web premium para **Villavicencio Biodigestores**, una empresa líder en Argentina con más de 10 años de experiencia en infraestructura sanitaria y energías renovables.

![Biodigestores Villavicencio Logo](public/favicon.svg)

## ✨ Características

- **Diseño Premium:** Estética moderna basada en *Glassmorphism* y animaciones sutiles.
- **Responsive Design:** Totalmente adaptado para móviles, tablets y escritorio.
- **Sección de Productos:** Detalle técnico de biodigestores, trampas de grasa y logística.
- **Preguntas Frecuentes:** Acordeón interactivo con información técnica clave.
- **Directorio Regional:** Mapa de contactos de todas las sucursales del país.
- **SEO Optimizado:** Estructura semántica lista para motores de búsqueda.
- **DarkMode Support:** Interfaz adaptable a las preferencias del usuario.

## 🚀 Tecnologías Utilizadas

- **Astro:** Framework de alto rendimiento para sitios orientados a contenido.
- **Tailwind CSS:** Para un diseño moderno y responsive.
- **Material Symbols:** Iconografía técnica y profesional.
- **JavaScript:** Interacciones fluidas y lógica de UI.

## 🛠️ Instalación y Desarrollo

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/tu-usuario/biodigestores-villavicencio.git
   cd biodigestores-villavicencio
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Ejecutar en modo desarrollo:**
   ```bash
   npm run dev
   ```
   El sitio estará disponible en `http://localhost:4321`.

## 📦 Despliegue en Vercel

Este proyecto está optimizado para **Vercel**:

1. Sube el código a un repositorio de GitHub.
2. Conecta el repositorio en el dashboard de Vercel.
3. El comando de build es `npm run build` y el directorio de salida es `dist/`.
